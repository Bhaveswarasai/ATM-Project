# 🏦 ATM Machine Simulation (Java)

A simple **ATM Machine Simulation Project** built using **Core Java**, demonstrating real-world banking operations and Object-Oriented Programming concepts.

---

## 🚀 Features

- 🔐 Secure PIN-based login system
- 💰 Deposit money
- 💸 Withdraw money
- 📊 Check account balance
- 🔄 Continuous transaction support
- ❌ Exit option

---

## 🛠️ Tech Stack

- Java (Core Java)
- OOP Concepts
- Scanner Class (for user input)

---
